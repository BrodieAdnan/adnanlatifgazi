Badge Reader

Badge Reader è una piattaforma embedded che permette di accedere ad un sistema elettronico convalidando badge assegnati a specifici utenti. Questo progetto, che ha richiesto un lavoro di gruppo, è formato una parte software e una hardware. Il software, reallizato in arduino, php, html, css, javascript e android, forma un sistema cloud-based che amministra l'hardware, riconosce e gestisce gli accessi utente; l'hardware effettua la lettura fisica e il riconoscimento dei badge quando avvicinati al sensore apposito.

Strong Box

Strong Box è una piattaforma embedded che simula le funzionalità di una cassaforte accessibile mediante codice di sicurezza. Il progetto, realizzato in team, è composto da una parte software (programma arduino avviato in locale) e una parte hardware (cassaforte il legno e sensori di inserimento codice di sicurezza). La componente software gestisce gli accessi alla cassaforte, mentre l'hardware legge il codice inserito e apre lo sportello.

Spread Sheet

Spread Sheet è una piattaforma interamente software che simula il funzionamento dei fogli elettronici in stile Microsoft Excel. Il sistema è stato realizzato in java, anche mediante l'utilizzo di una libreria grafica appositamente creata per facilitare la gestione front-end del software. Permette l'inserimento e la gestione dati mediante operazioni matematiche e logiche. Il motivo della sua creazione è puramente dovuto alla volontà di mettere a prova le proprie abilità di realizzazione e gestione di un progetto, nonchè di progettazione di un sistema complesso.

IFYT

IFYT (I Fix It for YouTube) è un progetto di gruppo ("I Fix It" è il nome del gruppo). IFYT è un sistema software che mette a dispozione diversi servizi YouTube all'utente in un'apposita pagina web, in modo che si possa far uso di tale funzioni in modo semplice e velce. Mediante un'interfaccia semplice e intuitiva, l'utente può cercare e riprodurre video, mettere like, fare log-in ... La piattaforma è stata realizzata usando java, javascript, html, css, web service sprignboot e API YouTube.

Bus Monitoring

Bus Monitoring è un simulatore di bus-tracker in tempo reale degli autobus del servizio ACTV. Ricrea una mappa virtuale della città, con strade e tratte: simula quindi lo spostamento dell'insieme degli autobus. La piattaforma, creata in un primo momento e ripresa e ampliata in un secondo momento, è realizzata in javascript, html, css e php: nell'ultima versione si utilizzano dati presi dalla banca ACTV per una simulazione reale sempre affidabile.
